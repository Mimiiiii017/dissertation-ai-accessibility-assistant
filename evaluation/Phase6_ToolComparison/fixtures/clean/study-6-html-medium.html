Frequently Asked Questions

How does the free trial work?

You get 14 days of full access to all Pro features, no credit card required. At the end of the trial you can choose a plan or your account reverts to the free tier.

Can I cancel at any time?

Yes. You can cancel your subscription from the account settings page. Your access continues until the end of the current billing period.

Is my data secure?

All data is encrypted at rest and in transit using AES-256 and TLS 1.3 respectively. We are ISO 27001 certified and undergo annual third-party security audits.

Do you offer discounts for non-profits?

Yes. Non-profit organisations receive a 50% discount on all plans. Contact our sales team with your registered charity number to apply.

What integrations are available?

We integrate natively with Slack, Microsoft Teams, Jira, GitHub, GitLab, Notion, Zapier, and over 200 other tools via our open REST API and webhook system.

Developer Documentation

Our REST API is fully documented with interactive examples. The client SDKs cover JavaScript/TypeScript, Python, Ruby, Go, and Java.

Quick Start

curl -X POST https://api.example.com/v1/events \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"event": "page_view", "properties": {"path": "/home"}}'

Authentication

All API requests must include a valid API key in the Authorization header as a Bearer token.

Rate Limits

The default rate limit is 1,000 requests per minute per API key. Enterprise plans support higher limits on request.

Pagination

List endpoints return a maximum of 100 items per page. Use the cursor parameter from the response's next_cursor field to fetch subsequent pages.

Webhooks

Configure webhook endpoints from Settings → Webhooks. Each event payload is signed with HMAC-SHA256 using your webhook secret.

Security & Compliance

We take the security of your data seriously. Here is an overview of the measures we have in place:

  • Encryption at rest: All stored data is encrypted using AES-256.
  • Encryption in transit: All network communication uses TLS 1.3 with HSTS enabled.
  • Access control: Role-based access control (RBAC) with fine-grained permissions at the resource level.
  • Audit logging: Every API call and user action is logged with actor, timestamp, and IP address.
  • Penetration testing: Annual third-party penetration tests conducted by CREST-certified firms.
  • ISO 27001: Certified since 2023; surveillance audits conducted bi-annually.
  • SOC 2 Type II: Report available under NDA for enterprise customers.
  • GDPR: Data Processing Agreements available; UK and EU data residency options.
